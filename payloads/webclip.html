WebClip¶

PayloadType
com.apple.webClip.managed
Name Type Title Description Required
PayloadDescription string Payload Description Description of the payload n/a
PayloadDisplayName string Payload Display Name Name of the payload always
PayloadIdentifier string Payload Identifier A unique identifier for the payload, dot-delimited. Usually root PayloadIdentifier+subidentifier always
PayloadType string Payload Type The type of the payload, a reverse dns string always
PayloadUUID string Payload UUID Unique identifier for the payload (format 01234567-89AB-CDEF-0123-456789ABCDEF) always
PayloadVersion integer Payload Version The version of the whole configuration profile. always
PayloadOrganization string Payload Organization This value describes the issuing organization of the profile, as displayed to the user n/a
URL string URL The URL that the web clip should open when clicked. The URL must begin with HTTP or HTTPS always
Label string Label The name of the Web Clip as displayed on the Home screen always
Icon data Icon A PNG icon to be shown on the Home screen. Should be 59 x 60 pixels. If not specified a white square will be shown n/a
IsRemovable boolean Removable If not set, the user cannot remove the Web Clip, but it will be removed if the profile is deleted n/a
FullScreen boolean Full Screen Launch the Web Clip as a full-screen web app n/a
Precomposed boolean Precomposed Icon Prevent SpringBoard from adding “shine” to the icon n/a
